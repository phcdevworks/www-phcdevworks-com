# Security Policy

## Reporting a Vulnerability

Please report security issues privately to the PHCDevworks maintainers through
GitHub Security Advisories or another private channel already established for
this repository.

Do not open a public GitHub issue for a suspected vulnerability.

## Response Expectations

- We will review the report and confirm receipt as soon as practical.
- We will assess impact, determine remediation steps, and coordinate disclosure.
- We will share resolution guidance once a fix or mitigation is ready.
